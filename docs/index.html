BronzeFit AI

Elderly Wellness Companion

This GitHub Pages site is the public entry page for BronzeFit. The full app is a live Next.js + MQTT project, so GitHub Pages cannot run the complete experience by itself.

Open GitHub Repository View Web App Source

Why Not Full GitHub Pages?

  • The app uses Next.js App Router and server API routes.
  • The exercise flow depends on a live MQTT bridge process.
  • The SIoT / UNIHIKER broker is a local-network dependency.

Recommended Deployment

  • Easiest: run the web app and MQTT bridge on one local always-on machine.
  • Public option: deploy the web app to Render or Railway.
  • Keep the MQTT bridge on the same network as the UNIHIKER if SIoT stays local.

Current BronzeFit Features

  • Exercise selection and session flow
  • Pose tracking and form scoring
  • UNIHIKER + SIoT MQTT proof-of-concept
  • Family dashboard and session summaries
