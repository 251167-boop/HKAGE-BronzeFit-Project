BronzeFit AI

Elderly wellness companion with exercise guidance, session scoring, and a UNIHIKER + SIoT MQTT proof-of-concept for live heart-rate integration.

This GitHub Pages entry is a lightweight static landing page. The full app itself runs as a Next.js project with API routes and a local MQTT bridge, so it still needs the project runtime to use all features.

Current PoC

UNIHIKER publishes fake BPM data to SIoT, the local bridge forwards it, and the exercise session UI displays live heart rate and warnings.

Local Run

Start the bridge with node mqtt-bridge.js, then run the web app in web/ with npm run dev.

UNIHIKER

Run python3 unihiker_publish_vitals.py on the UNIHIKER with SIoT enabled to publish heart-rate test data.
